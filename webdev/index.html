Technology

  • HTML5

HTML: Hypertext Markup Language

  • Hypertext: links documents together
  • Markup language: annotate text to describe document hierarchy and semantics
  • HTML5: new functionality like <video> and <canvas>

CSS: Cascading Style Sheets

  • Describe presentation of documents (layout and formatting)
  • Uses selectors
  • Cascading: more specific selectors override less specific ones
  • Style sheets: a list of selectors

JavaScript

  • Feature sparse
  • Interpreted
  • Prototype-based
  • Dynamic typing
  • First class functions
  • Client and server-side
  • Not without its warts

	header.innerHTML = 'Hello moon!';
}
</script>
</head>
<body>
<h1 id="header">Hello world!</h1>
</body>
</html>
